## Як використовувати цей шаблон

1. **ChatGPT**: вставляй шаблон і проси ChatGPT заповнити `{{ }}` на основі твого опису задачі
2. **Вручну**: копіюй, заповнюй секції, вставляй у Claude
3. **Мінімальна версія**: якщо задача мала — достатньо Кроку 0 (cgc/chub вивід) + Кроку 4 (опис) + Кроку 5 (верифікація)

## Ключові принципи

- **cgc/chub завжди першими** — без контексту Codex дає гіршу якість
- **Codex Gate завжди** — навіть якщо всі NONE, заповни і зафіксуй
- **Verification завжди** — жодного "готово" без доказів
- **Skills перед реалізацією** — не після
